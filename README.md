# Olist Power BI Dashboard

Dashboard analítico desenvolvido em Power BI utilizando o dataset público da Olist (e-commerce brasileiro).

O projeto consolida múltiplas visões de negócio em um único painel interativo, permitindo análise exploratória e acompanhamento de métricas operacionais e comerciais.

---

## Visões disponíveis

- Produtos  
- Pedidos  
- Pagamentos  
- Vendas  
- Vendedores  
- Avaliações  

Cada seção foi estruturada para responder perguntas práticas de negócio e facilitar interpretação visual.

---

## Principais análises

O dashboard permite investigar, entre outros pontos:

- Distribuição e volume de vendas
- Comportamento de pedidos ao longo do tempo
- Dinâmica de pagamentos e meios utilizados
- Performance e participação de vendedores
- Padrões em frete e valores médios
- Tendências temporais (mês, ano, acumulados)
- Comportamento e classificação de avaliações

---

## Dataset

Fonte pública amplamente utilizada em estudos de analytics:

**Brazilian E-Commerce Public Dataset by Olist**

Os dados incluem informações sobre pedidos, clientes, produtos, vendedores, pagamentos, fretes e reviews.
